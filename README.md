## Repository

In GitHub, repository refers to a storage space where a project is kept. A repository contains all the files related to a project, such as source code, documentation, and other resources, along with a history of changes made to those files. It allows developers to manage, track, and collaborate on projects efficiently, whether the repository is public for anyone to view or private for restricted access.

### Purpose of Repository

The purpose of a repository is to upload Assignment 1 of Development Operations module and to store all related files in one place. It also serves as a version control platform to track updates, manage changes, and ensures that the latest version of assignment 1 is always available if any modifications are made.
